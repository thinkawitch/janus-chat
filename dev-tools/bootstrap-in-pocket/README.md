# Bootstrap in pocket
Tool to place bootstrap in the container (pocket, holder).


```bash
npm install
```



---
old 
### install original bootstrap
```bash
cd bootstrap-in-pocket
git clone https://github.com/twbs/bootstrap.git ./bootstrap
cd bootstrap
git checkout tags/v5.1.3 -b andrew-app
npm install
```

### make css
```bash
cd bootstrap-in-pocket
npm --prefix bootstrap run css
```
